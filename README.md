# Файлы для итогового задания

В директории `tests` находятся тесты для проверки API, которое должно быть реализовано в веб-сервере.

Директория `web` содержит файлы фронтенда.

Описание проекта
Todo-приложение — это веб-сервер на Go с REST API для управления задачами. Приложение позволяет создавать, редактировать, удалять и отмечать задачи как выполненные. Поддерживается функциональность повторяющихся задач с различными правилами повторения.

Задания повышенной трудности (со звёздочкой)
Реализация всех правил повторения
Реализация поиска по дате или слову в задаче
Реализация возможности определять извне порт при запуске сервера
Реализация возможности определять путь к файлу базы данных через переменную окружения

Требования
Go 1.25.1 и выше
SQLite

Установка и запуск
1) Клонирование репозитория:
git clone -b https://github.com/Maxagena/final.git
cd final

2) Установка зависимостей:
go mod download

3) Запуск сервера:
go run main.go

4) Откройте в браузере:
http://localhost:7540

5) Запуск тестов:

настройска параметров в tests/settings.go

 var Port = 7540 
 var DBFile = "../scheduler.db" 
 var FullNextDate = false // поменять на true
 var Search = false // поменять на true
 var Token = `` // значение токена, которое сервер возвратит из /api/signin и которое хранится в куке token

Запустить все тесты

go test -v ./tests
